Noir supports various programming languages and frameworks. You can check the provided list with the command `noir --list-techs`.

## Available Support Scope
### Endpoint's Entities
- Path
- Method
- Param
- Header
- Cookie
- Protocol (e.g ws)
- Details (e.g The origin of the endpoint)
